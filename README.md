- 站点：https://cissy0802.github.io/personal-finance/
- 与 **investing** 仓分工：investing 讲「怎么让钱增值」，本站讲「怎么管好、守住、规划自己的钱」。
- 风格：可执行协议（数字 / 决策框架 / 检查表 / 本周一试），不荐股、不预测。默认美国账户与税制，中美差异处标注。
- 每期一条「给科技高薪人员的提示」签名线（相关才写）。
- 路线图见 `TOPICS.md`（routine 只读不改）；生成规范见 `ENGINE.md`；内容层自治见 `CLAUDE.md`。
